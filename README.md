## 実行コマンド

```bash
# 開発環境
$ yarn dev

# ビルド
$ yarn build

# ビルドしたプロジェクトの立ち上げ
$ yarn start

# コードチェック
$ yarn lint

# コードフォーマッタ
$ yarn format
```
